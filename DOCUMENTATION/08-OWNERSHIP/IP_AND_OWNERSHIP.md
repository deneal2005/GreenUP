## 7. COPYRIGHT & LICENSE

### Source Code License
**Current**: Public GitHub repository (MIT License implied, not explicitly stated)

**Options**:
1. **MIT License** (Recommended)
   - Permissive (allows commercial use, modification)
   - Requires attribution
   - No liability
   - Good for open-source / attracting contributors

2. **Apache 2.0**
   - Similar to MIT
   - Includes patent grant
   - Slightly more protective

3. **Proprietary / Closed Source**
   - Would require changing GitHub to private
   - Not recommended for open-source positioning
   - Limits community contributions

**Recommendation**: Add LICENSE file with MIT license to GitHub repository (already public)

```
MIT License

Copyright (c) 2026 GreenUP

Permission is hereby granted, free of charge, to any person obtaining a copy...
```

**Include in Terms of Service**:

to use, display, and share your content on the platform and in impact reports.

**Risk**: Violate open-source license terms (e.g., fail to attribute)
**Impact**: Copyright infringement claim, forced license change
**Likelihood**: Low (using permissive licenses)
**Mitigation**:

**Risk**: Future developer claims ownership of contributed code
